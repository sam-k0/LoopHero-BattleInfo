# BattleInfo

Mod to show detailed stats during battle.

## Settings

Contributes the following settings to `options.ini`:

```ini
[BattleInfo]
; Decimal precision for stats (eg. 1.23)
DecPrecision=2
; Offset for the text in px
TextOffsetX=0
TextOffsetY=0
; Font size for the text
TextSize=0.500000
```

## Dependencies
- LoopHeroCallbackCore
